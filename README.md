## 🎯 구현 목표

- 숫자 버튼을 눌러서 숫자 입력
- 연산자 버튼으로 사칙연산 선택
- `=` 버튼을 누르면 계산 결과 출력
- `AC` 버튼으로 전체 초기화

---

## 🛠️ 사용한 기술

- React (useState 사용)
- JSX 기반 컴포넌트 구성
- 간단한 CSS 스타일링
